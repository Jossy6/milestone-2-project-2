# Newsweek Clone

A responsive Newsweek website clone built with html and minimal external CSS, primarily using Bootstrap for layout and styling.

## Features
- Responsive design for desktop, tablet, and mobile views
- Minimal external CSS for styling adjustments
- Bootstrap grid system for layout management
- Sections include top stories, featured news, and trending topics

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## Installation & Usage
1. Clone this repository:
   ```sh
   git clone https://github.com/your-username/newsweek-clone.git
   ```
2. Navigate to the project directory:
   ```sh
   cd newsweek-clone
   ```
3. Open `index.html` in your browser to view the project.

## Customization
- Modify `styles.css` for additional custom styling.
- Update `index.html` to add or change content.
